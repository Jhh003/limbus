## ⚠️ 常见问题

### Q: 排行榜页面现在是空的，这是正常的吗？
**A**: 完全正常！因为还没有CloudBase的数据库，所以没有数据显示。

### Q: 我能不能现在就试试提交数据？
**A**: 可以，但会失败。现在的流程是：
```
用户 → 前端 → 等待后端API 😴
```

需要部署CloudBase才能完成这个链路。

### Q: 数据会不会丢失？
**A**: 不会！使用CloudBase的话，数据存在腾讯云的数据库里，非常安全。

### Q: 需要多少钱？
**A**: 个人使用完全免费！腾讯云给的免费额度足够。

```
免费额度：
✓ 云数据库：1GB/月
✓ 云函数：400万调用/月  
✓ CDN：10GB/月
✓ 对象存储：5GB/月

足够个人项目用！
```

### Q: 可以先不做认证系统吗？
**A**: 完全可以！这就是Phase 1的MVP方案。

```
Phase 1 (现在)
└─ 用户匿名提交，不需要登录

Phase 2 (以后)
└─ 添加用户认证和个人主页
```
